## How to Run

### Option 1: Open Directly in Browser
1. Clone or download this repository
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)
3. That's it! The application is ready to use.

### Option 2: Using a Local Server
```bash
# If you have Python installed:
python -m http.server 8000

# Or if you have Node.js with npx:
npx serve

# Then open http://localhost:8000 in your browser
```

### Option 3: Using VS Code Live Server
1. Install the "Live Server" extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

## Usage
1. **Browse Art Gallery**: Scroll to the gallery section to view artworks
2. **Shop Apparel**: Navigate to the store section to browse products
3. **Filter Products**: Use category buttons to filter by type
4. **Add to Cart**: Click "Add to Cart" on any product
5. **View Cart**: Click the shopping cart icon in the header
6. **Checkout**: Click the checkout button to complete your order
